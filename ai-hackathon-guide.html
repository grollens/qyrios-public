Det här handlar inte om verktyg

Ett hackathon för att upptäcka nya sätt att tänka med AI. Inte nya knappar att trycka på.

Du väljer själv vad du utforskar och hur djupt du går. Det enda som krävs är nyfikenhet.

Scrolla ner

Vi är inte gifta med någon teknik

AI-landskapet rör sig snabbt. Det som dominerar idag var inte ens relevant för några månader sedan, och det som kommer härnäst vet ingen säkert. Vi testar och lär oss längs med vägen — och just nu är Claude-sviten en av de klarast lysande stjärnorna när det gäller att gå från ”använda AI” till ”samarbeta med AI.”

Målet med det här hackathonet är inte att lära sig ett specifikt verktyg. Det är att upptäcka ett nytt arbetssätt — så att när nästa verktyg kommer, vet du redan hur du tänker.

Begrepp att känna till

Prompt grundläggande

Instruktionen eller frågan du ger till AI. Kan vara allt från en mening till en detaljerad brief.

Kontext grundläggande

Agent AI-begrepp

En AI som inte bara svarar på frågor utan kan planera, fatta beslut och utföra flera steg autonomt. Både Cowork och Code använder agentbeteende.

MCP tekniskt

Model Context Protocol — en öppen standard för att koppla AI till externa verktyg och datakällor som Airtable, Google Drive, Slack med mera. Tänk det som USB-portar för AI.

Skill Claude

En förbyggd förmåga i Cowork/Code — t.ex. att skapa Excel-filer, bygga dashboards, eller analysera data. Claude väljer rätt skill automatiskt baserat på din brief.

Artifacts Claude

Interaktiva leverabler som Claude skapar i chatten — kod, webbsidor, diagram eller dokument som du kan se och använda direkt i konversationen.

LLM AI-begrepp

Large Language Model — den typ av AI-modell som driver Claude, ChatGPT och liknande. Tränad på enorma mängder text för att förstå och producera språk.

01 — Kärnan

Fyra paradigmskiften

De flesta använder AI som en sökmotor med bättre svar. Det fungerar — men det är som att använda en smartphone bara för att ringa. Det intressanta händer när du börjar förhålla dig annorlunda — när du slutar ställa frågor och börjar samarbeta.
1
Chat

Från googling till tankepartner

Ställa en fråga, få ett svar → Tänka högt tillsammans

De flesta stannar vid fråga–svar. Men redan i chatten kan du skifta: be Claude utmana dina antaganden, ställ följdfrågor på svaret, be om alternativa perspektiv. Använd AI som en samtalspartner som hjälper dig tänka bättre — inte bara svara snabbare.

2
Projects

Från repetition till delad förståelse

Förklara allt varje gång → Bygga gemensam kontext

Istället för att börja från noll i varje konversation, ger du Claude ert sortiment, era riktlinjer, er data. Nu förstår Claude er värld — och svaren blir dramatiskt bättre. Det är skillnaden mellan att prata med en främling och en kollega.

3
Cowork

Från frågor till delegering

”Vad tycker du om denna data?” → ”Analysera datan och skapa en rapport.”

Här händer det stora skiftet. Du slutar använda AI som ett verktyg och börjar briefa det som en medarbetare. Du beskriver önskat resultat — Claude planerar stegen, exekverar dem autonomt, producerar leverabler, och stämmer av med dig när det behövs. Det är som att delegera en komplex uppgift till en kollega: du säger vart ni ska, inte hur ni tar er dit. Claude söker på webben, analyserar filer, skriver kod i bakgrunden, och levererar färdiga dashboards, rapporter och presentationer.

4
Code

Från uppgifter till system

Lösa en sak i taget → Bygga något som fortsätter arbeta

Code tar den autonoma förmågan ett steg längre. Precis som Cowork planerar och exekverar Code komplexa uppgifter självständigt — men här bygger du något som fortsätter arbeta efter att du stängt laptopen. En agent som dagligen bevakar konkurrenter, en pipeline som transformerar data, ett gränssnitt som hela teamet kan använda. Du väver in AI i hela tankeprocessen: från ”hjälp mig med detta” till ”bygg något som gör detta åt oss, varje dag.”

↻

Iteration över perfektion

Din första prompt behöver inte vara perfekt. Starta messy, iterera, förfina. AI belönar ”good enough” som startpunkt — magin ligger i dialogen som följer.

↑

Metalärande

Den viktigaste färdigheten är inte att promptra — det är att lära sig lära med AI. Ställ följdfrågor. Ifrågasätt svaren. Be om fördjupning. Varje interaktion lär dig vad AI kan och hur du använder det bättre.

04 — Upplägg

2 timmar. Fritt utforskande.

Kort intro, sedan utforskar du själv — med handledare i rummet som hjälper dig komma igång och gå vidare.

1
0:00 — 0:15 (15 min)

Kick-off & mindset

Kort intro om paradigmskiftena: varför vi är här, vad som är möjligt, och vad som skiljer ”använda AI” från ”samarbeta med AI”. Snabb demo av ett och samma problem löst i Chat vs. Cowork för att visa skillnaden i arbetssätt. Översikt av mock-databasen och tillgängliga verktyg.

2
0:15 — 1:40 (85 min)

Utforska & experimentera

Välj ett verktyg som känns rätt för dig. Jobba ensam eller i par. Utgå från ett problem du känner igen från vardagen, eller välj en av de föreslagna utmaningarna nedan. Handledare cirkulerar för att hjälpa dig komma igång, föreslå nästa steg, och visa vad som är möjligt. Byt gärna verktyg under vägen — det är ingen tävling, det är en upptäcktsresa.

3
1:40 — 2:00 (20 min)

Show & tell

Kort runda där den som vill visar vad de hittade, byggt eller blivit överraskade av. Det mest värdefulla är ofta insikterna, inte leverablerna. Avsluta med: vad tar du med dig? Vad vill du testa imöron?

Det finns inget ”rätt” sätt

Du behöver inte kunna koda. Du behöver inte ha en plan. Det enda vi ber dig om är att prova något du inte gjort förut — och se vad som händer.

06 — Förberedelser

Vad behöver vi göra innan?

Praktisk checklista för att allt ska rulla från minut ett.

🎯 Konton & licenser

Minimum: Pro-plan ($20/mån per person)
Ger tillgång till Chat, Projects, Cowork och Code. Användningen är delad mellan alla verktyg med en 5-timmarscykel som återställs. För en 2h-workshop räcker Pro för de flesta.

Rekommenderat för Code-spåret: Max-plan ($100/mån)
5x mer användning än Pro. Om några deltagare vill bygga intensivt med Code kan de annars slå i taket på Pro.

Alternativ för team: Team-plan ($25/mån per person)
Ger admin-kontroll över alla konton, gemensam fakturering, och enklare onboarding. Bra om ni vill ha kontroll över 10+ deltagare.

Tips: Skapa konton i förväg och verifiera att alla kan logga in. Skicka inloggningsuppgifter dagen innan.

💻 Teknisk setup

Chat & Projects — Ingen installation. Fungerar i webbläsaren på claude.ai.

Cowork — Kräver Claude Desktop-appen (Mac/Windows). Installera och testa i förväg. Cowork aktiveras i desktopappen.

Code — Kräver terminal och CLI-installation. Se till att deltagare som vill använda Code har installerat det innan workshopen.

Wi-Fi — Säkerställ stabil internetuppkoppling för alla deltagare. AI-verktyg är bandbreddskrävande.

📊 Data & material

Mock-databas — Förbered realistisk retaildata: försäljning per butik/kategori, kundrecensioner, lagernivåer, konkurrentpriser. Gör tillgänglig som CSV-filer, enkel API, och/eller Airtable-bas.

Förberett Project — Skapa ett Claude Project med relevant kontext (sortimentsdata, varumärkesriktlinjer) så deltagare kan utforska Projects-funktionen direkt.

Delat dokument — Förbered en enkel guide med länkar till alla resurser, inloggningsuppgifter och föreslagna utmaningar som deltagarna kan ha öppet under sessionen.
